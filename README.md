# Health Data Visualization Dashboard

## Overview
A lightweight, responsive frontend dashboard designed to visualize mock medical and genomic datasets. This project focuses on the intersection of healthcare and user interface design, translating complex data into clean, readable visual metrics. 

## Why I Built This
Coming from a foundational background in pre-medical sciences and transitioning into tech, I realized that data is only as useful as how well it can be understood. I built this dashboard to practice structuring clean frontend architecture while applying advanced graphic design principles to clinical data presentation.

## Tech Stack
* **HTML5:** Semantic structure and layout.
* **CSS3:** Custom clinical color palette (no external CSS frameworks like Bootstrap). Built with Flexbox/CSS Grid for responsive behavior.
* **JavaScript (Vanilla):** DOM manipulation and logic.
* **Chart.js:** Data visualization (Line and Bar charts).

## Features
* **Real-time Charting:** Renders patient vitals (simulated heart rate) and sample comparisons using Chart.js.
* **Clinical UI/UX:** Utilizes a precise, high-contrast color scheme optimized for readability in a medical context.
* **Modular Codebase:** Hardcoded data arrays in `script.js` are clearly separated, making it easy to plug in a real backend API or database later.

## How to Run
This is a purely frontend application. No local server or build tools are required.
1. Clone the repository.
2. Open `index.html` in any modern web browser.
